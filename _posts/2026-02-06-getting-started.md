Deze setup stelt je in staat om Obsidian te gebruiken als je blog post editor, terwijl Jekyll je website genereert.

## Waarom deze setup?

- **Obsidian**: Geweldige markdown editor met linking, tags en graph view
- **Jekyll**: Krachtige statische site generator
- **Samen**: Het beste van beide werelden!

## Workflow

Je workflow is simpel:

1. Schrijf posts in Obsidian
2. Sla ze op in de `_posts` folder
3. Run Jekyll om je site te bouwen
4. Deploy naar GitHub Pages of je eigen hosting

## Obsidian features die je kunt gebruiken

- **Internal Links**: `[[Andere Notitie]]` (deze moeten wel geconverteerd worden voor Jekyll)
- **Tags**: Gebruik de front matter tags
- **Templates**: Maak post templates in Obsidian
- **Daily Notes**: Kunnen hergebruikt worden als blog posts
